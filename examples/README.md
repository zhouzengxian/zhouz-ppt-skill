# Examples

这里存放示例，帮助你快速理解如何使用 zhouz-ppt-skill。

## 快速体验

1. 复制 `../assets/template/{index.html,style.css,app.js}` 到本目录。
2. 复制 `demo-slides.js` 为 `slides.js`。
3. 双击 `index.html` 预览（或 `python -m http.server 8000` 后访问 `http://localhost:8000/`）。

## demo-slides.js

一份涵盖封面、章节、目录、数据、卡片、时间线、对比、引用、结束的完整示例幻灯片数组，可以直接作为 `slides.js` 使用，也可以作为你编写自己幻灯片的模板。

阅读 [../references/slide-types.md](../references/slide-types.md) 了解每种类型的完整 JSON schema。
